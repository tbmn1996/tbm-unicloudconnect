Willkommen bei UniCloudConnect

Dieses Tool lädt ausgewählte LearnWeb-Inhalte lokal auf deinen Mac und kann Aufzeichnungen lokal als Markdown transkribieren. Keine Konfigurationsdateien, kein Terminal.

Read-only gegenüber LearnWeb
Die App liest nur — sie verändert nichts in LearnWeb.
Alles bleibt lokal auf deinem Mac
Keine Cloud, kein Notion. Du entscheidest, wo die Dateien liegen.
Zugangsdaten sicher in der Keychain
Nie im Klartext, nie online, nie in Logs oder Exporten.
Läuft sichtbar im Hintergrund
Über die macOS-Menüleiste immer erreichbar.

MCP erlaubt Claude oder Codex, read-only auf dein LearnWeb zuzugreifen — solange UniCloudConnect läuft. Das ist optional und wird nie still aktiviert.

Wichtig zu verstehen: MCP gewährt kontoweiten read-only Zugriff auf dein LearnWeb — nicht begrenzt auf deine Sync-Auswahl.
Nur Lesezugriff — Claude kann nichts in LearnWeb ändern.
